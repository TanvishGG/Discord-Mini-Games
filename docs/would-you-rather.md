# Would You Rather
Get a random Would You Rather question.
## Importing

```js
 const {WouldYouRather} = require('discord-mini-games.js')
```
## Getting a Would You Rather question.

```js
 const wyr = WouldYouRather();
 console.log(wyr)
```
 - **output:**

```
Would you rather be blind or deaf?
```
